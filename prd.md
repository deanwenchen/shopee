  - 表单输入背景 #f8f8f8

### 4. Password（密码输入页面 - 多状态合并）

> **注意**: 此页面已合并 PasswordTyping 和 WrongPassword 的功能，通过 `inputMode` 状态控制显示不同的 UI。

#### 🎯 功能目标
统一的密码输入页面，支持 4 位输入 → 8 位输入 → 错误重试 三种状态。

#### 📝 功能描述
- **状态 1 (4-digit)**: 显示 4 个空输入框占位，输入任意字符后切换到 8 位模式
- **状态 2 (8-digit)**: 显示 8 个密码点（已输入蓝色实心，未输入灰色空心），输满 8 位后自动验证
- **状态 3 (error)**: 显示 8 个红色实心点（错误状态），提供 "Forgot your password?" 链接

#### 📐 业务规则
- 输入第 1 个字符 → 切换到 8 位密码模式
- 输满 8 位 → 验证密码
  - 密码正确 (`12345678`) → 跳转至 HelloCard
  - 密码错误 → 切换到错误状态（红点）
- 错误状态下退格 → 红点变蓝点，允许重新输入
- 错误状态下输满 8 位错误密码 → 蓝点变红点（确认错误）
- 点击 "Not you?" → 返回登录页面（仅在非错误状态）
- 点击 "Forgot your password?" → 进入密码重置流程（仅错误状态）

#### 🎨 UI 结构
```
┌─────────────────────────────┐
│      StatusBar              │
│       💙 Bubbles BG         │
│                             │
│      [Avatar Circle]        │
│                             │
│   Hello, Romina!!           │
│                             │
│   Type your password        │
│                             │
│  [4 个空框 或 8 个密码点]      │
│                             │
│  Not you? → (非错误状态)     │
│  Forgot your password? (错误状态)│
│                             │
│      [Home Indicator]       │
└─────────────────────────────┘
```

## 完整交互逻辑规格

### 1. 密码输入流程（4 位 → 8 位）

#### 流程说明
```
Password (4 空框)
    ↓ 输入任意字符
    ↓ (自动跳转)
PasswordTyping (8 蓝点)
    ↓ 输满 8 位
    ↓ (自动验证)
    ├── 正确 (12345678) → HelloCard
    └── 错误 → WrongPassword (8 红点)
```

#### Password 页面交互
- 4 个空框占位
- 输入任意字符后自动跳转
- "Not you?" 按钮返回登录

#### PasswordTyping 页面交互
- 密码长度：固定 8 位
- 视觉反馈：蓝色实心点 (已输入) / 灰色空心点 (未输入)
- 自动聚焦输入框
- 密码验证时机：输满 8 位后延时验证
- 正确密码：`12345678`

#### WrongPassword 页面交互
- 密码长度：固定 8 位
- 视觉反馈：红色实心点 (已输入) / 灰色空心点 (未输入)
- 输满 8 位后返回登录
- "Forgot your password?" 跳转密码恢复

```
                                                                                                                                        ├── 正确 ──→ NewPassword
                                                                                                                                        │                       │
                                                                                                                                        │                       └── Save ──→ Login
                                                                                                                                        │
                                                                                                                                        └── 错误 3 次 ──→ MaximumAttempts Popup
                                                                                                                                                              │
                                                                                                                                                              └── Okay ──→ Login
```

### 8. 数据验证规则汇总

| 页面 | 字段 | 规则 | 错误提示 |
|------|------|------|----------|
| LoginPage | Email | 必填，格式验证 | "Please enter your email" / "Please enter a valid email address" |
| LoginPage | Email | 账号存在检查 | "Account not found. Please check your email." |
| CreateAccount | Email | 必填，格式验证 | "Please enter your email" / "Please enter a valid email address" |
| CreateAccount | Password | 必填，≥8 位 | "Please enter a password" / "Password must be at least 8 characters" |
| PasswordTyping | 密码 | 8 位，固定值 `12345678` | 跳转 WrongPassword |
| PasswordRecoveryCode | 验证码 | 4 位，固定值 `1234` | 计数 +1，3 次后显示弹窗 |
